# Image Tool
## _For Image Processing_

Image tool has 14 diffrent functions.

-  Blur Image
- Deblur Image
- Grayscale Image
- Crop Image
- Flip Image
- Mirror Image
- Rotate Image
- Reverse Colors
- Add Noise
- Edge Detection
- Change Color Balance(RGB)
- Increase Brightness
- Adjust Contrast
- Increase Saturation

And it has load and save buttons.
